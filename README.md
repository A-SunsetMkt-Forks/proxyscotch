## Building
*These build scripts are for macOS/Linux systems. Currently, Windows build scripts have not yet been created.*

- For *NIX (Linux/macOS) systems:
```bash
$ ./build.sh unix
```

- For Windows systems:
```bash
$ ./build.sh win32
```
